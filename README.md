## 運行環境要求

- Windows 7及以上
- PHP 7.1+
- MySQL 5.7.7+


## 開發環境佈署／安裝

本項目代碼使用 php 框架 laravel 8.* 開發，開發環境為 Laravel Homestead。
下文將假設已經是在 Homestead 環境建立好的情況下說明。如還未安裝請參考 [Homestead 安裝與設置](https://laravel.com/docs/8.x/homestead) 進行配置與安裝

### 基礎安裝
#### clone 源代碼
clone 源代碼到本地：
    > git clone
